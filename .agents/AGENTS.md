# Antigravity Rules

## Mode: Autonomous / Direct Execution
- Work autonomously and apply changes directly without asking for confirmation for every single action.
- Only ask questions or request confirmation when strictly necessary (e.g., for destructive, dangerous, or irreversible operations).
- Assume continuation of the current workflow and proceed proactively.
- Bias for action: Less questions, more action.
- Confirmation level: Low.
